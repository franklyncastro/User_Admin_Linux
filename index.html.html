Guía de Práctica - Administración de Usuarios y Permisos Linux

Introducción

Esta guía está diseñada para practicar la administración de usuarios, grupos, permisos y seguridad básica en Linux. Los ejercicios están organizados por secciones para ayudarte a desarrollar habilidades reales utilizadas por administradores e ingenieros Linux.

Recomendación: Realiza todos los ejercicios en una máquina virtual para evitar afectar tu sistema principal.

Contenido

1 - Administración de Usuarios 2 - Administración de Grupos 3 - Permisos de Archivos 4 - Permisos en Directorios 5 - ACL (Permisos Avanzados) 6 - Escenarios Reales

Sección 1 - Administración de Usuarios

Ejercicio 1 - Crear Usuario Básico

Objetivo: Crear un usuario llamado ana.
sudo useradd -m -s /bin/bash ana sudo passwd ana id ana

Ejercicio 2 - Crear Usuario sin Login

sudo useradd -m -s /usr/sbin/nologin backupsvc cat /etc/passwd | grep backupsvc

Ejercicio 3 - Crear Múltiples Usuarios

sudo useradd -m carlos sudo useradd -m maria sudo useradd -m pedro

Ejercicio 4 - Bloquear Usuario

sudo passwd -l maria sudo passwd -u maria

Ejercicio 5 - Expirar Cuenta

sudo chage -E 2026-12-31 pedro sudo chage -l pedro

Sección 2 - Administración de Grupos

Ejercicio 1 - Crear Grupo

sudo groupadd developers getent group developers

Ejercicio 2 - Agregar Usuario a Grupo

sudo usermod -aG developers ana groups ana

Ejercicio 3 - Compartir Grupo

sudo usermod -aG developers carlos sudo usermod -aG developers maria

Ejercicio 4 - Remover Usuario del Grupo

sudo gpasswd -d maria developers

Ejercicio 5 - Grupo Administrativo

sudo groupadd sysadmins sudo usermod -aG sysadmins ana

Sección 3 - Permisos de Archivos

Ejercicio 1 - Permisos Básicos

touch prueba.txt chmod 644 prueba.txt ls -l prueba.txt

Ejercicio 2 - Script Ejecutable

touch backup.sh chmod 755 backup.sh

Ejercicio 3 - Quitar Permisos

chmod o-r prueba.txt

Ejercicio 4 - Permisos Simbólicos

chmod u+x backup.sh chmod g-w backup.sh

Ejercicio 5 - Archivo Privado

chmod 600 secreto.txt

Sección 4 - Permisos en Directorios

Ejercicio 1 - Directorio Privado

mkdir privado chmod 700 privado

Ejercicio 2 - Directorio Compartido

mkdir compartido chmod 770 compartido

Ejercicio 3 - Grupo Propietario

sudo chown :developers compartido

Ejercicio 4 - SetGID

chmod g+s compartido

Ejercicio 5 - Sticky Bit

chmod +t compartido

Sección 5 - ACL (Permisos Avanzados)

Ejercicio 1 - Instalar ACL

sudo apt install acl

Ejercicio 2 - Asignar ACL

sudo setfacl -m u:carlos:rwx secreto.txt

Ejercicio 3 - Ver ACL

getfacl secreto.txt

Ejercicio 4 - Eliminar ACL

sudo setfacl -x u:carlos secreto.txt

Ejercicio 5 - ACL por Defecto

sudo setfacl -d -m g:developers:rwx compartido
